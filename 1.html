HTML이란 무엇인가?

Hyper Text Markup Language (HTML) is the standard markup language for creating web pages and web applications. Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
